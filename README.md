# javascript-lotto-precourse

### 프로젝트 개요 및 설명

간단한 로또 발매기를 구현할 것이다.
사용자는 로또 구입 금액을 입력해 1000원짜리 로또를 1개 이상 구매할 것이다. 프로그램은 구매한 각 로또 종이에 중복하지 않는 6개의 숫자를 랜덤으로 뽑아 채울 것이다.

여기서 사용자가 당첨 번호와 보너스 번호를 입력하면 프로그램은 수익률을 계산해준다.

---

### 주요 기능

- 로또 종이 구입
- 각 종이에 중복하지 않는 6개의 숫자를 랜덤으로 지정해주기
- 당첨 번호와 보너스 번호 입력받기
- 각 종이의 당첨 여부와 몇 등인지 확인하기
- 수익률 계산

---

### 기능 목록 작성

에러 발생 시 [ERROR]로 시작하는 메세지와 함께 Error을 발생시키고 해당 메세지를 출력 후 그 지점부터 다시 입력을 받는다.

- [x] 로또 구매하기
  - 정상 경우
    - 입력 받은 숫자를 1000으로 나눠 구매한 로또 종이 개수(n) 구하기
  - 예외 경우
    - 숫자가 아닌 것을 입력했을 경우
    - 1000으로 나뉘어지지 않은 것을 입력했을 경우
- [x] 각 종이에 중복하지 않는 6개의 숫자 랜덤으로 지정해주기
  - 정상 경우
    - MissionUtils.Random.pickUniqueNumbersInRange(1,45,6)을 사용해서 1~45 범위의 숫자 중에 6개의 숫자를 뽑는다.
    - ascending order로 sort해준다
    - 이를 로또 종이 개수인 n번 만큼 반복하고 배열의 배열에 저장해준다.
- [x] 당첨 번호와 보너스 번호 입력 받기
  - 정상 경우
    - 당첨번호 입력 받기
    - 당첨번호 입력 받은 것을 쉼표로 split하기
    - 보너스 번호 입력 받기
  - 예외 경우
    - 당첨번호 입력받은 것을 쉼표로 split했는데 각 element가 숫자가 아닌 경우
    - 각 element가 숫자이긴 하나 범위 (1~45) 사이의 숫자가 아닌 경우
    - 번호가 6개가 아닌 경우
    - 반복되는 숫자가 존재하는 경우
    - 보너스 번호가 숫자가 아닌 경우
    - 보너스 번호가 범위 (1~45) 사이의 숫자가 아닌 경우
    - 보너스 번호가 앞선 6개의 숫자에 이미 포함되어 있을 경우
- [x] 각 종이의 당첨 여부와 몇 등인지 확인하기
  - 정상 경우
    - 배열의 배열 속의 값들을 보고 당첨번호 & 보너스 번호와 비교하기
    - 각 등수에 몇개 나왔는지 기록하기
- [ ] 수익률 계산하기
  - 정상 경우
    - ((당첨된 금액 - 로또 사는데 쓴 돈)/로또 사는데 쓴 돈) \* 100
    - 소수점 둘째 자리에서 반올림하기
- [ ] indent 3개 이상 되어있는 것들 줄이기
- [ ] Jest로 테스트 만들기
